  - 测试集：`(10000, 4000)`，已标准化

## 关键参数配置

### 数据参数
```python
# 输入维度
IN_CHANNELS = 1      # 单导联ECG
ECG_LENGTH = 4000    # 信号长度

# 任务划分
SSL_SAMPLES = 19000  # 无标签样本
POSITIVE_SAMPLES = 500  # AF样本
NEGATIVE_SAMPLES = 500  # Non-AF样本

# 先验概率
POSITIVE_PRIOR = 0.0256  # π = 500/(500+19000)
PI_PRIME = 0.5          # 目标平衡先验
```

### 训练参数
```python
# SSL预训练
SSL_EPOCHS = 100
LR_SSL = 1e-3
BATCH_SIZE = 32

# NNPU微调
NNPU_EPOCHS = 50
LR_NNPU = 1e-4
LAMBDA_PU = 5.0      # PU损失权重
TEMPERATURE = 0.1    # SSL温度参数
```
